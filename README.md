## Installation

Windows & MacOS:

```sh
npm install
```

Ubuntu:

```sh
sudo npm install
```

## Usage example

Após baixar e instalar os node_modules, seu aplicativo está pronto pra ser iniciado.

Back-end:
_With [yarn][yarn]._
```sh
yarn dev
```
_No yarn_
```sh
npm dev
```

Front-end:
```sh
npm start
```

Mobile:
```sh
npm start
```
_Ou_
```sh
expo start
```
